# 🧭 十二方位中心定位工具 (Image Compass Aligner)

一個輕量、直覺且完全執行於瀏覽器端的網頁工具。使用者可以透過簡單的「貼上」或「拖曳」圖片，利用畫面中央永遠置頂的紅色虛線 **十二地支（子丑寅卯）方位儀**，進行精準的中心點對齊與各象限的方位辨識。

---

## ✨ 核心功能

* **📋 快捷貼上即用**：支援系統剪貼簿，直接在網頁按下 `Ctrl + V` (或 Mac 的 `Cmd + V`) 即可瞬間載入截圖。
* **🔴 顯眼置頂方位線**：12 方位分隔線與中心十字統一採用**高飽和度鮮紅色虛線**，永遠疊加在圖片最上層，在任何複雜的地圖或圖表背景下皆清晰可見。
* **🖱️ 自由平移與對齊**：載入圖片後，可直接在畫面上使用滑鼠點擊並拖曳圖片，輕鬆將目標主體對準中心基準點。
* **🔍 側邊直向縮放拉桿**：畫面右側設有懸浮縮放面板，往上拉放大、往下拉縮小，同時支援**滑鼠滾輪**以畫面中心點為基準進行細緻縮放。
* **📱 雙端支援**：完全相容桌上型電腦滑鼠操作與智慧型手機的觸控拖曳。

---

## 🚀 使用說明

1.  **開啟網頁**：使用瀏覽器打開 `compass_tool.html`。
2.  **載入圖片**：複製任意圖片或截圖後，在網頁任意處按下 `Ctrl + V`。
3.  **調整位置**：按住滑鼠左鍵（或單指觸控）拖曳圖片，將你想觀測的中心點對準紅色的中心十字。
4.  **調整大小**：使用滑鼠滾輪，或是操作右側的縮放拉桿（➕ / ➖），放大或縮小圖片以進行精確比對。
5.  **辨識方位**：以中心點為準，輻射出去的 12 個青銅金色圓圈即為對應的「子、丑、寅、卯...」等十二地支方位。

---

## 🛠️ 技術特點

* **單一檔案 (Single File)**：純 HTML5 + Vanilla JavaScript 編寫，無須安裝任何後端環境或伺服器，雙擊即可執行。
* **極速繪製 (Canvas API)**：方位羅盤與線條使用 HTML5 Canvas 動態繪製，確保在高解析度螢幕下依然銳利不失真。
* **現代化 UI (Tailwind CSS)**：導入 Tailwind CSS 打造簡潔、美觀的深色系（Slate）操作介面，讓視覺焦點集中在圖片與方位上。
